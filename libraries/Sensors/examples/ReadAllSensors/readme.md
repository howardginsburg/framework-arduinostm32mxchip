# ReadAllSensors

Reads all onboard sensors (temperature, humidity, pressure, accelerometer, gyroscope, magnetometer) and displays the values on the OLED screen and Serial monitor.

## Hardware

- MXChip AZ3166 DevKit (all sensors are onboard)

## Running

1. Open the Serial Monitor at 115200 baud.
2. Sensor readings update every 2 seconds on both OLED and Serial.
